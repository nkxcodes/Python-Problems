# Python Problems

This repository contains my Python programming practice problems.  
The goal of this repo is to improve my logic building and problem-solving skills step by step.

---

## Purpose

Programming is not just about writing code. It is about thinking in steps and building logic that a computer can follow.

This repo helps me:
- Strengthen basic programming logic
- Practice solving problems consistently
- Improve from beginner level to advanced level over time
- Track my learning progress

---

## Structure

The repository is divided into three levels:

### Beginner
Basic problems to understand fundamental concepts like:
- Conditions (if/else)
- Loops
- Basic input/output
- Simple number operations

### Intermediate
Problems that require more thinking and combination of concepts:
- Functions
- Patterns
- Math-based problems
- String manipulation

### Advanced
More complex problems focused on:
- Recursion
- Algorithmic thinking
- Optimization
- Problem solving under constraints

---

## How I Use This Repo

For each problem, I follow this process:

1. Understand the problem clearly
2. Write the logic in simple English steps
3. Convert the steps into Python code
4. Test with different inputs
5. Improve the solution if possible
